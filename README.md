# ✨ MindSpark

**Votre collection personnelle de citations inspirantes.**

MindSpark est une application web légère qui permet de collecter, organiser et redécouvrir les citations qui vous inspirent au quotidien.

## Fonctionnalités

- **Citation à l'honneur** — une citation mise en avant au chargement, renouvelable d'un clic
- **Ajout rapide** — sauvegardez vos propres citations en un instant
- **Recherche instantanée** — retrouvez une citation par mot-clé
- **Statistiques** — aperçu de votre collection
- **Gestion complète** — suppression, réinitialisation, citation aléatoire

## Installation

Clonez le dépôt et installez les dépendances :

```bash
pip install -r requirements.txt
```

## Lancement

```bash
streamlit run app.py
```

L'application sera accessible sur `http://localhost:8501`.

## Prérequis

- Python 3.8 ou plus récent
- pip
